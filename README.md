# Heart Attack Risk Classification

## Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction of heart attack risk can help in taking preventive medical actions.

This project builds a **machine learning classification model** that predicts whether a person is at risk of a heart attack based on clinical and lifestyle attributes.

The project includes **data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation** using Python and machine learning libraries.

---

## Objectives

* Analyze medical data related to heart health
* Identify important features affecting heart attack risk
* Build and train machine learning classification models
* Evaluate model performance using appropriate metrics
* Predict heart attack risk for new data

## Dataset

The dataset contains medical and lifestyle attributes that influence heart health, such as:

* Age
* Cholesterol
* Blood Pressure
* Heart Rate
* Lifestyle factors
* Other clinical measurements

The target variable indicates **whether a person is at risk of a heart attack or not**.
